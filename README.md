# 📊 Customer Churn Analysis Dashboard

## 📌 Overview

This project presents an interactive **Customer Churn Analysis Dashboard** built using Power BI.
The goal is to analyze customer behavior and identify key factors that contribute to customer churn.

---

## 🎯 Objectives

* Understand **customer churn rate**
* Identify **high-risk customer segments**
* Analyze patterns across:

  * Geography
  * Gender
  * Age
  * Credit Score
  * Account Balance
  * Product Usage
  * Activity Status

---

## 📂 Dataset

The dataset contains customer information including:

* Customer ID
* Geography
* Gender
* Age
* Credit Score
* Balance
* Number of Products
* IsActiveMember
* Exited (Churn indicator)

---

## 📊 Key Metrics

* **Total Customers:** 10,000
* **Churn Customers:** ~2,000
* **Churn Rate:** ~20%

---

## 📈 Dashboard Features

The dashboard includes:

* **KPI Cards**

  * Total Customers
  * Churn Customers
  * Churn Rate

* **Visual Analysis**

  * Churn by Geography
  * Churn by Gender
  * Churn by Age
  * Churn by Tenure
  * Churn by Number of Products
  * Churn by Active Status
  * Churn by Credit Score Group
  * Churn by Balance Group

---

## 🧠 Key Insights

* Customers in **Germany and France** show higher churn compared to Spain
* **Female customers** churn more than male customers
* Customers aged **40–60** have higher churn rates
* **Low credit score** customers are more likely to churn
* Customers with **higher balance** show significant churn behavior
* **Inactive customers** churn more than active ones
* Customers with **fewer products (especially 1)** are more likely to leave
